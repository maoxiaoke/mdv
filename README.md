# mdv

TypeScript + Node.js starter.

## Quick start

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Scripts

- `npm run dev` тАФ run in watch mode (tsx)
- `npm run build` тАФ build to `dist/`
- `npm start` тАФ run the built output
- `npm run lint` тАФ eslint
- `npm run format` тАФ prettier
- `npm test` тАФ vitest
